Get Started

UP AND RUNNING
IN 60 SECONDS

No installers. No accounts. No subscriptions.

1
Download the files
Download fss_b110_complete.html, fss_bridge.py and FSS_START.bat into the same folder on your PC.
2
Install Python (once)
Download Python 3.10+ from python.org. During install, tick "Add Python to PATH". You only need to do this once.
3
Double-click FSS_START.bat
The launcher installs irsdk and websockets automatically, starts the bridge, and opens Chrome at the right URL. No command line needed.
4
Connect and drive
Go to the Live tab and click Connect. Start a session in iRacing — your engineer comes online automatically. Or load an .ibt file in the Telemetry tab for offline analysis.
