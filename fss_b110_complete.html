Changelog
B 1.10STABILITY + FIXES -- Fixed page freeze caused by orphaned top-level code from old corner detection function. Added missing HTML body tag. Removed conflicting demo animation setInterval. Imola track map locked at correct 90deg orientation. Voice coaching silenced in demo/offline mode. Live tyre temps honour iRacing C/F preference. Menger curvature corner detection (pure geometry). Tyre temp state machine fires on transition only. Dashboard 8/8 module count protected by explicit ID.
B 1.09MODULE 08 LIVE — Live AI Voice Engineer. Python WebSocket bridge (fss_bridge.py) reads iRacing shared memory at 20Hz. 5 engineer personas (Performance/Qualifying/Endurance/Driver Coach/Minimal). Push-to-talk speech recognition. Natural phrase engine with non-repetition logic. Fuel/pit/tyre/traffic/mistake triggers. Adaptive silence rules. Post-session replay summary.
B 1.08MODULE 07 LIVE — Practice Engine. Pre-session planner with lap time target, consistency target, clean laps target, focus areas, session notes. Live timestamped session log with tags (Note/Good/Issue/Setup/Lap). Telemetry cross-reference auto-runs on IBT load — Plan vs Actual on every target. Export log to .txt. Persists in localStorage.
B 1.07MODULE 05 LIVE — Setup Engine. IBT CarSetup YAML extraction. Tyre temp/wear analysis. Qualifying, Stable and Aggressive setup variants. STO file download for each variant.
B 1.06MODULE 04 LIVE — AI Driver Coach. Brake trace & consistency analysis. Throttle application profiling. Corner-by-corner entry/apex/exit breakdown. Steering smoothness & sawing detection. Traffic management & disrupted lap identification. AI coaching report via Claude API with personalised improvement priorities.
B 1.06VERSION UPDATE — All dashboard module cards, nav badge, build version, and footer updated to B 1.06. Module 2 sections now collapse on every new IBT load. Map Source hidden from Track Map status bar.
B 1.06MODULE 03 CLEAN RESET — Removed all external API, iRacing login, and OSM code. GPS reconstruction from IBT Lat/Lon (dense stride=1 re-extraction). Loop closure + PCA auto-rotate. Speed heatmap, brake, throttle overlays. No corners. No login. Auto-loads on IBT load.
B 1.06STABLE CORNER ZONES — Replaced unstable peak-splitting with clean zone-based detection. One continuous above-threshold region = one corner. 70th-percentile adaptive threshold. Min zone length filter removes noise. Single retry at lower multiplier if <4 corners found. Stable: threshold changes no longer cause corner count instability.
B 1.06CORNER SPLIT ENFORCEMENT — Multi-peak detection: each curvature peak in a zone is its own corner. Valley >40% drop = split. Menger curvature replaces heading-delta. Auto-retry at lower threshold until min corner count met. Over-detect preferred. tmFindPeaks separates chicanes.
B 1.06GEOMETRY PRESERVATION — Removed uniform resampling (linear interp was rounding 90° corners). Disabled tmSmooth entirely. Raw GPS points used directly (~4500 at 60Hz). Corner detection windows scale to point density. Shape deviation validator added. Lat/Lon GPS is the correct source (WorldPosX/Y does not exist in iRacing IBT).
B 1.06CORNER APEX FIX — Apex = max instantaneous curvature (Menger radius), not min speed. Adaptive curvature threshold. Chicanes produce separate markers (no forced merging). Label offset outward from track via normal vector with leader line. Preserves 90° street circuit geometry.
B 1.06TRACK MAP PIPELINE — Full data pipeline: sort by LapDistPct, continuity validation, 1000-pt uniform resample via interpolation. Geometry curvature corner detection (direction-change rate, not inputs). Canvas size reduced. XY teleport clamping.
B 1.06TRACK MAP ACCURACY — Dense full-resolution lap re-extraction (stride=1, ~4500 samples vs 55). Loop closure via linear drift correction. PCA auto-rotation to optimal orientation. Clockwise winding normalisation.
B 1.06MODULE 03 LIVE — Track Map Engine. IBT WorldX/Y geometry reconstruction with dead-reckoning fallback. Speed heatmap, brake & throttle overlays. Data-driven corner detection T1–Tn with entry/apex/exit markers. Straight detection S1–Sn. Chicane merging. Full segmentation table.
B 0.11TYRE + LAP DETAIL — Heat/Pressure blank tab fix. °C/°F toggle. Lap Detail: all channels stacked. Wear consumption bar chart. Version auto-stamping from FSS_VER constant.
B 0.10TYRE ANALYSIS MODULE — Heat, Wear, Pressure, Slip Angle. Lap-by-lap + Trace views. 4-corner diagram. Wear consumption delta per lap. JS syntax freeze fix.
B 0.09YAML fixes — DriverCarIdx player lookup for car/driver. TrackAirTemp/TrackSurfaceTemp/TrackWindVel correct field names. Detected Channels completely removed.
B 0.086 bug fixes — multi-channel overlay chart, Module 02 dashboard card live, tick rate/samples hidden, metadata cleaned.
B 0.07MODULE 02 LIVE — IBT Telemetry Engine. Direct .ibt binary load. Multi-channel overlay chart. Lap deltas. Corner segmentation.
B 0.06COMPLETE SCHEDULE — All GT3 series added (Sprint, Regional x3, Fixed), GTE Sprint, LMP2 Challenge, TCR Virtual x2, NASCAR iRacing Series (mirrors real-world Cup), SLM Tour, NASCAR Modified Tour. 42 total series. All Road, Formula, Oval verified.
